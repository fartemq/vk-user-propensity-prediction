# vk-user-propensity-prediction
Разработка и обучение модели машинного обучения, которая по совокупности структурированных и текстовых данных сможет предсказать склонность пользователя к целевому действию

# Моделирование склонности пользователей VK к целевому действию
## Описание проекта
Этот репозиторий содержит решение аналитического кейса компании **VK**. Цель проекта — разработать продвинутый pipeline машинного обучения, который предсказывает склонность пользователя совершить критически важное для бизнеса целевое действие.

Проект решает классическую, но сложную задачу мультимодального анализа данных: объединение зашумленных структурированных табличных признаков с неструктурированными текстовыми логами.

## Бизнес-цель и задачи
* **Ценность для бизнеса:** Точное определение целевой аудитории позволяет VK оптимизировать расходы на таргетированную рекламу, улучшить онбординг новых пользователей и повысить общую конверсию в продукт.
* **Главная задача:** Обучить ML-модель классификации, превосходящую базовые метрики, используя совмещенный набор данных (таблицы + текст).

## 🛠️ Технологический стек
* **Язык программирования:** Python
* **Обработка данных:** Pandas, NumPy
* **Машинное обучение:** Scikit-learn, LightGBM / CatBoost
* **Обработка текста (NLP):** TF-IDF / Embeddings
* **Среда разработки:** Jupyter Notebook

## 📋 План реализации проекта
1. **Разведочный анализ данных (EDA) и предобработка:** Анализ дисбаланса классов, природы пропусков в данных и первичный осмотр текстовых полей.
2. **Проектирование и отбор признаков (Feature Engineering):** Извлечение признаков из логов, генерация эмбеддингов для текста и отбор наиболее значимых фичей.
3. **Обучение моделей:** Сравнение нескольких алгоритмов (от логистической регрессии до градиентного бустинга), настройка гиперпараметров и борьба с дисбалансом.
4. **Валидация и оценка качества:** Оценка финального качества модели на отложенной выборке с помощью метрик ROC-AUC и F1-score.
